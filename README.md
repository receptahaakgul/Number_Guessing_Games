## Number_Guessing_Game
**This game is a number guessing console game made within Unity.**

### Steps:

+ Extract the file
+ Open Unity Hub.
+ Add the folder to Unity Hub and run it.
+ Start the game.
+ Guess your number.
+ Follow the instructions given in the Console window.
